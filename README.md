# Bookstore_SQLProject
This project is based on a Bookstore Database, designed to simulate a real-world bookstore system. It covers core database concepts such as table relationships, data insertion, and business-oriented SQL queries.

# Database Description
The database represents a bookstore with information about:
•	Authors
•	Books
•	Customers
•	Orders
•	Order items

# SQL Files
1. create_tables.sql
Contains SQL statements to create the database schema
2. insert_data.sql
Contains sample data insertion for all tables
3. business_queries.sql
Contains SQL queries that answer business-level questions
Focused on Sales analysis, Customer behavior and Product performance
Uses JOIN, GROUP BY, COUNT, SUM, AVG, and sorting techniques

# Project answers key business questions such as:
1.Top selling books

2.Top customers by number of orders or revenue

3.Revenue by genre

4.Customers with repeat orders

5.Sales performance by day
